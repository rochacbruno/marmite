## About this Site

This site was built with the **Marmite Playground**, an interactive environment
for learning and experimenting with [Marmite](https://marmite.blog).

## About Marmite

Marmite (**Mar**kdown **M**akes S**ite**s) is a minimal static site generator
written in Rust. It takes a folder of Markdown files and produces a complete
website - no configuration required.

**Key features:**

- Single binary, zero runtime dependencies
- Zero-config by default, fully customizable via `marmite.yaml`
- Built-in themes and colorschemes
- Syntax highlighting at build time (no JavaScript)
- Math (KaTeX) and diagrams (Mermaid) support
- RSS feeds, sitemaps, and search
- Built-in dev server with live reload
