# Special Characters in Tags

This post demonstrates that tags with special characters work correctly.

The tag `Comunicação` should generate a file named `tag-comunicacao.html` (NOT `tag-comunicac-a-o.html`).

The same applies to other tags with special characters like:
- Programação
- Documentação

This ensures consistency between the tag archive filenames and the tag links generated in templates.
